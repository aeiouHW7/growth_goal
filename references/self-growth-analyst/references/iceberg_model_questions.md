# 冰山模型追问序列 (Iceberg Model Questions)

## 7层结构
1. **行为 (Behavior)** - 表面可观察的行动
2. **应对方式 (Coping)** - 如何应对情境
3. **感受 (Feeling)** - 当时的情绪
4. **观点 (Perception)** - 当时的想法
5. **期待 (Expectation)** - 希望发生什么
6. **渴望 (Longing)** - 深层需求
7. **自我 (Self)** - 身份/价值认同

---

## 触发关键词 → 起始层

| 关键词 | 起始层 |
|--------|--------|
| "搞砸了"、"失败了"、"没做" | 行为层（第1层） |
| "真累"、"受不了"、"撑不住" | 感受层（第3层） |
| "我觉得"、"我想"、"我认为" | 观点层（第4层） |
| "如果能...就好了"、"要是...就太好了" | 期待层（第5层） |
| "我想要..."、"我需要..."、"我渴望..." | 渴望层（第6层） |
| "我是一个...的人"、"我就是..." | 自我层（第7层） |

---

## 逐层追问序列

### 从行为（第1层）开始

**起点问句**: "你做了什么？"

**追问第2层（应对）**: "你应对这个情况的方式是什么？"
- 如果是回避 → "你用什么方式回避？"
- 如果是攻击 → "你做了什么来应对？"
- 如果是顺从 → "你是怎么配合的？"

**追问第3层（感受）**: "你当时的感受是什么？"
- 情绪词：愤怒/沮丧/恐惧/兴奋/羞耻/愧疚/羞耻...
- 如果说不清 → "你现在身体有什么感觉？"
- 如果模糊 → "你说'不好受'，具体是什么？是失望？沮丧？还是愤怒？"

**追问第4层（观点）**: "你在想什么？"
- 引导："你对这个情况的判断是什么？"
- 深层："这个判断背后，你相信了什么？"

---

### 从感受（第3层）开始

**起点问句**: "你现在的感受是什么？"
- 情绪粒度："你说'不好受'，具体是什么？"

**追问第2层（应对）**: "你怎么处理这种感受？"
- 如果是逃避 → "你用什么方式让自己不去感受？"
- 如果是发泄 → "你通过什么方式把情绪释放出来？"
- 如果是压抑 → "你会把这种感觉压下去吗？"

**追问第4层（观点）**: "这种感受告诉你什么？"
- "你觉得这件事意味着什么？"

---

### 从观点（第4层）开始

**起点问句**: "你在想什么？"

**追问第5层（期待）**: "你希望事情怎么发展？"
- "你期望的结果是什么？"
- "你觉得应该怎样才对？"

**追问第6层（渴望）**: "你真正想要的是什么？"
- "这个期待背后，你真正需要的是什么？"
- "如果这个期待被满足了，你会怎样？"

---

## 深层追问路径（从第6层到第7层）

**追问第7层（自我）**: "如果你获得了这个渴望，对你意味着什么？"
- "你是一个怎样的人？"
- "你的价值建立在什么之上？"
- "如果没有这件事，你是谁？"

---

## 追问规则

### 黄金法则
1. 每次只追一层，不追多层
2. 用"什么"开头，不用"为什么"
3. 等用户回答，再追问下一层
4. 不假设用户会防御，允许沉默
5. 不给建议，只提问

### 禁止事项
- ❌ "你为什么这样做？"（太抽象、太逼迫）
- ❌ "你到底在想什么？"（逼迫感）
- ❌ 直接跳到第7层（太快）
- ❌ "你应该..."（给建议）
- ❌ "我理解你，因为..."（打断用户）

### 允许沉默
用户沉默时，耐心等待，不要急于填补空白。如果沉默过长，可以轻轻问："你想说什么？"

---

## Coach联动输出

当冰山追问完成后，输出结构化诊断：

```json
{
  "trigger_keyword": "失败",
  "entry_layer": 1,
  "path_followed": [1, 2, 3],
  "surface_findings": ["行为：抱怨领导", "应对：策略性服从", "感受：挫败感"],
  "deep_discovery": "渴望被认可",
  "coaching_focus": "帮助用户找到不依赖领导认可的自我价值感"
}
```

### 字段说明
- `trigger_keyword`: 触发追问的关键词
- `entry_layer`: 从哪一层开始追问
- `path_followed`: 追问经过的层级序列
- `surface_findings`: 各层的发现（简述）
- `deep_discovery`: 深层渴望
- `coaching_focus`: 下一步Coach引导方向

---

## 使用时机
- 当用户说"失败"、"沮丧"、"搞砸了"时 → 从行为层开始
- 当用户重复同样的行为模式时 → 从行为或应对开始
- 当干预效果不好时（深层原因未找到）→ 从感受层开始
- 当用户说"我觉得..."但没有行动时 → 从观点层开始
- 当用户说"我想要..."但一直得不到时 → 从期待层追到渴望层

---

## 各层典型干预方向

| 层级 | 干预重点 |
|------|----------|
| 第1层 行为 | 行为替换、行动实验 |
| 第2层 应对 | 觉察旧模式、探索新应对 |
| 第3层 感受 | 情绪命名、情绪陪伴 |
| 第4层 观点 | 认知重构、换个视角 |
| 第5层 期待 | 期待检验、现实调整 |
| 第6层 渴望 | 满足渴望的其他路径 |
| 第7层 自我 | 身份确认、价值重塑 |
