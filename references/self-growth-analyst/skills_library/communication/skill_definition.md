## 触发条件

|------|-----|------|
| pattern | 表达不清晰自知 | ≥2次 |

**触发机制**：任一条件满足即进入 diagnosis_flow，不等待多个条件同时满足。

## Diagnosis Flow

### Step 1: 冰山定位

- **方法**: iceberg
- **问题**: "你没说清楚，是因为没想清楚，还是想清楚了但表达不出来？"
- **输出**: 思维问题 vs 技巧问题
- **判断依据**:
  - 用户说"我也不知道想表达什么" → 思维问题
  - 用户说"我知道想说什么，但说出来就乱了" → 技巧问题

### Step 2: Fogg 模型诊断

- **方法**: fogg_model
- **问题**: "你现在会主动练习表达吗？"
- **输出**: 动机/能力/提示
- **判断依据**:
  - 不练说"没时间/没机会" → 提示缺失
  - 想练但做不到 → 能力问题
  - 觉得没必要练 → 动机问题

## 干预方案

### Type: micro_action

**Prompt**:
```
你知道自己表达不清晰（4/19承认过，4/24又承认），但没有任何改变。
这不是能力问题，是你没有把它当成必须解决的事。
```

**Action**:
> 下次开会前，把核心观点用一句话写下来，问自己"听众会问什么问题"
