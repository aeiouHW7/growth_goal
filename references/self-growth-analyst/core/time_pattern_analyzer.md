# Time Pattern Analyzer — 时间模式分析引擎

## 引擎类型
L2 按需触发引擎 — 不是每次碎碎念都跑，当触发条件满足时执行。

## L2触发条件
- 消息时间在 00:00-06:00（凌晨）
- 消息时间在 22:00-24:00（深夜）
- 用户说"累了"/"困了"/"熬夜晚"

## 时间状态推断表

| 时间段 | 精力状态 | 可能意图 |
|--------|---------|---------|
| 06:00-09:00 | 刚起床/清醒 | 正常 |
| 09:00-12:00 | 上午工作 | 执行中 |
| 12:00-14:00 | 午休 | 休息 |
| 14:00-18:00 | 下午工作 | 执行中 |
| 18:00-20:00 | 晚间收尾 | 低强度 |
| 20:00-22:00 | 晚间放松 | 低强度 |
| 22:00-00:00 | 深夜 | 透支/逃避 |
| 00:00-06:00 | 凌晨 | 亢奋/失眠/逃避 |

## 星期→行为模式表

| 星期 | 默认模式 | 放纵系数 |
|------|---------|---------|
| 周一 | 工作驱动 | 1.0 |
| 周二 | 工作驱动 | 1.0 |
| 周三 | 工作驱动 | 1.0 |
| 周四 | 工作驱动 | 1.0 |
| 周五 | 工作+放松 | 1.2 |
| 周六 | 周末 | 1.5 |
| 周日 | 周末准备 | 1.3 |

## 输出格式

```json
{
  "timestamp": "04:30",
  "time_bucket": "凌晨(00:00-06:00)",
  "time_inferred_state": "亢奋/失眠/逃避",
  "energy_level": "low",
  "day_of_week": "周六",
  "day_pattern": "周末放纵模式",
  "放纵系数": 1.5,
  "boundary_violation": true,
  "risk_indicators": ["睡眠不足风险", "凌晨决策质量低"]
}
```

## 干预建议生成

| 模式 | 干预建议 |
|------|---------|
| 凌晨(00:00-06:00) + 执行低 | "凌晨4点，你还在工作。身体已经在报警。" |
| 周末放纵 | "周六不是借口。放纵也要有边界。" |
| 工作日深夜 | "工作到深夜≠高效。明天早上你会付出代价。" |

## 与retrieval_index的联动

当 boundary_violation=true 时，在retrieval_index中查找：
- "计划被打断就跳过" 模式
- "知道但不做" 模式

作为干预参考。
