# 战略对齐引擎 (Strategic Alignment Engine)

## 核心目标
追踪用户当前行动与"成为中国前100富人"大目标的对齐程度。

## 分析维度
1. 时间投资：每天/每周花在"成长"上的时间占比
2. 方向聚焦：当前做的事情是否在一条主线上
3. 资源分配：精力/金钱/注意力是否优先给高价值事情
4. 技能积累：做的事情是否在建立稀缺能力

## 用户当前状态（从历史数据推断）
- 主线：大厂PM + AI工具开发 + 公众号（银发经济）
- 矛盾：工作占太多时间 → 公众号/成长时间被压缩
- 风险：多方向并行，缺乏聚焦

## 输出格式
```json
{
  "alignment_score": 1-10,
  "main_activities": ["工作", "AI工具开发", "公众号探索"],
  "alignment_to_goal": {"工作": "短期生存", "AI工具": "技能积累", "公众号": "长期杠杆"},
  "recommended_rebalance": "xxx"
}
```

## 运行指令

### 大目标定义
用户的终极目标：「成为中国前100富人」

这个目标的实现路径分析：
- 需要：稀缺技能 × 可规模化产品 × 资本积累
- 可能路径：大厂高收入 → 技能稀缺性 → 创业/投资 → 财富规模化
- 关键杠杆：AI工具开发（技能稀缺性）+ 公众号（个人品牌杠杆）+ 投资/创业（财富规模化）

### 分析维度与评分

**维度1: 时间投资（25%权重）**
收集用户过去一周的时间分配：
- 工作时间（生存必需，但过多会挤压成长时间）
- 成长时间（学习、技能建立、认知提升）
- 被动探索时间（看视频、刷社交等低价值活动）

评分规则：
- 成长时间占比 ≥ 30% → 维度得分 8-10
- 成长时间占比 20-30% → 维度得分 5-7
- 成长时间占比 < 20% → 维度得分 1-4

**维度2: 方向聚焦（25%权重）**
分析用户当前活动是否在一条主线上：

用户主要活动线：
- 工作线：大厂PM（当前收入来源，短期生存）
- 技能线：AI工具开发（建立稀缺能力）
- 杠杆线：公众号-银发经济（个人品牌 + 可规模化产品）

评分规则：
- 三条线形成合力，互相增强 → 得分 8-10
- 三条线各自独立，没有明显冲突 → 得分 5-7
- 三条线互相争夺时间/精力，分散 → 得分 1-4

**维度3: 资源分配优先级（25%权重）**
分析用户精力/注意力/金钱是否优先给高价值事情：

高价值事情：
- AI技能学习（稀缺能力）
- 公众号内容创作（个人品牌）
- 战略性人脉维护
- 投资学习与实践

低价值事情：
- 被动消费内容（刷视频等）
- 无目的社交
- 重复性行政工作

评分规则：
- 高价值事情占用主要资源 → 得分 8-10
- 高低价值事情时间各半 → 得分 5-7
- 低价值事情占用大部分资源 → 得分 1-4

**维度4: 技能积累（25%权重）**
分析做的事情是否在建立稀缺能力：

稀缺能力清单：
- AI工具开发能力（大厂PM + AI = 稀缺组合）
- 内容创作与个人品牌
- 产品化能力
- 投资与资本运作

评分规则：
- 每天都在积累稀缺能力 → 得分 8-10
- 部分时间积累，部分消耗在低价值活动 → 得分 5-7
- 主要在做消耗性工作，没有积累 → 得分 1-4

### 综合对齐评分计算

```
alignment_score = 时间投资得分×0.25 + 方向聚焦得分×0.25 + 资源分配得分×0.25 + 技能积累得分×0.25
```

评分解读：
- 8-10分：高度对齐，继续保持
- 5-7分：中等对齐，存在优化空间
- 3-4分：低度对齐，需要重新配置
- 1-2分：几乎不对齐，需要重大调整

## Coach 联动说明

将诊断结果传递给 Coach，Coach 生成战略调整对话：
- alignment_score < 5 时，重点讨论重新配置优先级
- alignment_score 5-7 时，讨论具体优化方案
- alignment_score > 7 时，讨论如何保持和加速
- main_gaps 中的每个问题生成一个追问，帮助用户找到自己的解决方案
