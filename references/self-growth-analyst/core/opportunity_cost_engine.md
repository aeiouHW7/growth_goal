# 机会成本分析引擎

## 章节1: 引擎目的

```
机会成本分析引擎
目的：分析用户"没做什么"带来的财富损失
原则：不评判，客观陈述
```

## 章节2: 活动分类

### 高杠杆活动（时间投入产出高）

- 学习核心技能（编程/产品/投资知识）
- 创作内容（公众号/视频/文章）
- 建立人脉（深度对话/资源链接）
- 投资理财（研究标的/配置资产）
- 健身运动（健康管理是长期杠杆）

### 低杠杆活动（时间投入产出低）

- 通勤（除非利用通勤时间学习）
- 无目的刷手机/短视频
- 无效社交（吃吃喝喝无产出）
- 纯消费娱乐（超过1小时/天）

### 空白时间（未记录的时间块）

- 不知道怎么花的时间
- 碎片时间（等电梯/排队）

## 章节3: 分析流程

从 daily_raw 提取一周数据：

1. 统计每个活动类别的小时数
2. 计算高杠杆 vs 低杠杆比例
3. 计算空白时间
4. 对比"理想时间分配"和"实际时间分配"

**理想分配（参考）**

- 高杠杆活动 >= 20小时/周
- 低杠杆活动 <= 10小时/周
- 空白时间 <= 5小时/周

## 章节4: ROI对比输出

```markdown
## 本周机会成本分析

### 时间分配
高杠杆：X小时
低杠杆：Y小时
空白：Z小时

### 机会成本
你本周在"低杠杆活动"上花了Y小时
如果把这Y小时投入"创作内容"，预估财富期望值提高Z

### 优先级建议
下周优先减少：X活动（从Y小时减到Z小时）
```

## 章节5: 调用时机

- 用户说"我很忙但没什么进展"时调用
