## ADDED Requirements

### Requirement: time_pattern_analyzer 引擎规格

**文件**: `core/time_pattern_analyzer.md`
**层级**: L2（按需触发，不是每次碎碎念都跑）
**Token预算**: ~150 tokens

#### L2触发条件
- 消息时间在 00:00-06:00（凌晨）
- 消息时间在 22:00-24:00（深夜）
- 用户说"累了"/"困了"/"熬夜晚"

### Requirement: 时间→状态推断

#### 时间状态表

| 时间段 | 精力状态 | 可能意图 |
|--------|---------|---------|
| 06:00-09:00 | 刚起床/清醒 | 正常 |
| 09:00-12:00 | 上午工作 | 执行中 |
| 12:00-14:00 | 午休 | 休息 |
| 14:00-18:00 | 下午工作 | 执行中 |
| 18:00-20:00 | 晚间 | 收尾 |
| 20:00-22:00 | 晚间放松 | 低强度 |
| 22:00-00:00 | 深夜 | 透支/逃避 |
| 00:00-06:00 | 凌晨 | 亢奋/失眠/逃避 |

#### Scenario: 凌晨发消息
- **WHEN** 消息时间在00:00-06:00
- **THEN** time_inferred_state = "凌晨状态：亢奋/失眠/或逃避现实"
- **AND** energy_level = "low"
- **AND** risk_indicators = ["睡眠不足风险", "决策疲劳"]

### Requirement: 星期模式分析

#### 星期→行为模式

| 星期 | 默认模式 | 放纵系数 |
|------|---------|---------|
| 周一 | 工作驱动 | 1.0 |
| 周二 | 工作驱动 | 1.0 |
| 周三 | 工作驱动 | 1.0 |
| 周四 | 工作驱动 | 1.0 |
| 周五 | 工作+放松 | 1.2 |
| 周六 | 周末 | 1.5 |
| 周日 | 周末准备 | 1.3 |

#### Scenario: 周六凌晨4:30睡
- **WHEN** 用户周六凌晨4:30睡觉
- **AND** time_pattern_analyzer判断为"周末放纵模式"
- **THEN** 输出：
  - day_pattern = "周末放纵模式"
  -放纵系数 = 1.5
  - boundary_violation = true（超过1点截止）

### Requirement: 输出格式

```json
{
  "timestamp": "04:30",
  "time_bucket": "凌晨(00:00-06:00)",
  "time_inferred_state": "亢奋/失眠/逃避",
  "energy_level": "low",
  "day_of_week": "周六",
  "day_pattern": "周末放纵模式",
  "放纵系数": 1.5,
  "boundary_violation": true,
  "risk_indicators": ["睡眠不足风险", "凌晨决策质量低"]
}
```
