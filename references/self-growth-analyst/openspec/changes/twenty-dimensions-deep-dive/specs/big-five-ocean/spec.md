## ADDED Requirements

### Requirement: 大五人格 OCEAN 底层分析引擎 (Big Five Ocean Analyzer)

基于大五人格模型（OCEAN）对用户进行长期、渐进式的心理特质建模，用于个性化干预策略的底层适配。

OCEAN 五维度定义：
- **O - 开放性 (Openness)** — 对新体验的接受度、好奇心、想象力
- **C - 尽责性 (Conscientiousness)** — 自律性、责任感、计划性
- **E - 外向性 (Extraversion)** — 社交活跃度、活力水平、刺激寻求
- **A - 宜人性 (Agreeableness)** — 同理心、合作性、信任他人
- **N - 神经质 (Neuroticism)** — 情绪不稳定程度、焦虑敏感度、抗压能力

---

#### Scenario: 数据积累期（前3个月）

- **WHEN** 系统运行未满3个月，或用户的有效数据点不足50条
- **THEN** 分析引擎仅执行数据记录模式：将每次观察到的行为模式归入对应OCEAN维度，不进行计算、不生成分析报告、不触发干预策略调整；所有数据存入 `big_five_data.md` 进行长期积累

---

#### Scenario: 计算启动——基线建立

- **WHEN** 系统运行满3个月且有效数据点达50条以上
- **THEN** 分析引擎启动基线计算，对每个OCEAN维度进行倾向性评估（高/中/低），输出格式：
```
{
  主导维度: <得分最高维度>,
  基线状态: {
    开放性: <高/中/低>,
    尽责性: <高/中/低>,
    外向性: <高/中/低>,
    宜人性: <高/中/低>,
    神经质: <高/中/低>
  },
  建议策略: <针对主导维度和异常维度的个性化干预建议>
}
```

---

#### Scenario: 尽责性低 —— 干预策略适配

- **WHEN** 基线计算结果显示`尽责性=低`
- **THEN** 系统自动调整干预策略：增加外部提醒频率、引入约束机制（如承诺书、公开承诺）、使用短周期目标拆分；建议Coach Agent减少依赖自我驱动的语言（如"你应该自律"），改用具体化、拆解化、可测量的行为提示

---

#### Scenario: 神经质高 —— 干预策略适配

- **WHEN** 基线计算结果显示`神经质=高`
- **THEN** 系统自动调整干预策略：强化情绪智力维度干预、降低高风险行为的建议频次、引入压力缓释机制；在用户处于高压状态时，优先触发情绪安抚而非目标推进；建议Coach Agent使用温和、支持性语言，避免批评和施压

---

#### Scenario: 季度基线更新

- **WHEN** 每90天累计新数据点达30条以上
- **THEN** 分析引擎重新计算基线，与上次基线对比，输出维度漂移报告；若某维度漂移超过一个等级（如尽责性从"中"变为"高"），在洞察中标注变化原因，供Analyst Agent和Coach Agent参考
