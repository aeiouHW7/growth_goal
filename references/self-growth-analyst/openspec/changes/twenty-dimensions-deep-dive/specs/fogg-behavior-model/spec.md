## ADDED Requirements

### Requirement: 福格行为模型诊断引擎 (Fogg Behavior Model — B=MAP)

当用户表现出"知道应该做但没有做"或"想做但没有行动"的行为模式时，系统激活福格行为模型诊断引擎，分析行为缺失背后的根本原因，并输出结构化的干预建议。

诊断输出结构：
```
{
  维度: <行为所属维度>,
  B不足: <动机不足 | 能力不足 | 提示不足>,
  具体原因: <分析说明>,
  建议干预: <对应的Coach Agent干预策略>
}
```

---

#### Scenario: 动机不足诊断

- **WHEN** 用户表达了想做某事的意愿，但持续没有行动，且语言中包含"想""应该""必须"等词汇，但没有相应行为跟进
- **THEN** 诊断输出 `B不足=动机不足`，具体原因聚焦于行为对用户的意义模糊或与深层价值脱节，建议Coach Agent通过价值澄清或微行为绑定来增强内在动机，而非施加外部压力

---

#### Scenario: 能力不足诊断

- **WHEN** 用户表达了想做某事但做不到，语言中透露"太难了""我不会""太复杂了"，或反复尝试失败后放弃
- **THEN** 诊断输出 `B不足=能力不足`，具体原因分析能力缺口的具体环节，建议Coach Agent降低行为难度（简化步骤、降低门槛、提供模板），而非催促或施压

---

#### Scenario: 提示不足诊断

- **WHEN** 用户表达了想做某事、有能力做，但没有任何触发行为的提示或线索出现（如没有提醒、没有环境触发、没有固定触发点）
- **THEN** 诊断输出 `B不足=提示不足`，具体原因指出当前环境中缺少触发因素，建议Coach Agent帮助用户设计行为提示（如时间提示、环境提示、社会提示）

---

#### Scenario: 复合不足诊断

- **WHEN** 用户的行为缺失同时满足两个或三个维度的不足条件
- **THEN** 诊断输出多个`B不足`项，按优先级排序（动机 > 能力 > 提示），建议Coach Agent优先处理最高优先级维度，同时在低优先级维度提供轻量辅助

---

#### Scenario: Coach Agent 联动——能力不足场景

- **WHEN** 诊断结果为`B不足=能力不足`
- **THEN** Coach Agent 的干预策略必须匹配：降低行为难度（如拆解任务、提供工具、减少步骤），明确禁止使用"你应该快点做""你怎么还不做"等催促式语言
