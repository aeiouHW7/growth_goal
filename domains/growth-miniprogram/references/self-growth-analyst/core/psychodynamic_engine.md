# 心理动力引擎 (Psychodynamic Engine)

## 冰山模型7层
1. 行为 (Behavior) - 表面可观察的行动
2. 应对方式 (Coping) - 如何应对情境
3. 感受 (Feeling) - 当时的情绪
4. 观点 (Perception) - 当时的想法
5. 期待 (Expectation) - 希望发生什么
6. 渴望 (Longing) - 深层需求
7. 自我 (Self) - 身份/价值认同

## 触发条件
当用户输入包含以下关键词时，触发冰山追问：
- "失败"、"搞砸了"、"后悔" → 从行为层往下追问
- "真的累了"、"受不了" → 从感受层追问
- "我觉得"、"我想" → 从观点层追问

## 追问规则
1. 每次只追一层，不追多层
2. 从行为往下挖，不跳层
3. 用"什么"不用"为什么"开头

## 运行指令

### 冰山层定义

```
层1-行为(Behavior): 用户做了什么/说了什么/如何反应
层2-应对方式(Coping): 用户如何应对这个情境/采取了什么策略
层3-感受(Feeling): 用户当时的情绪是什么
层4-观点(Perception): 用户当时在想什么/有什么想法/如何解读
层5-期待(Expectation): 用户希望发生什么/期望什么结果
层6-渴望(Longing): 用户的深层需求是什么/真正想要的是什么
层7-自我(Self): 用户如何定义自己/这反映了什么身份认同
```

### 触发检测

分析用户输入，检测关键词匹配：

| 触发词 | 起始层 | 说明 |
|-------|-------|------|
| 失败/搞砸了/后悔/完了/搞砸 | 层1-行为 | 从可观察行为开始追问 |
| 真的累了/受不了/撑不住了/心累 | 层3-感受 | 从情绪层开始追问 |
| 我觉得/我想/我认为/我的看法 | 层4-观点 | 从想法层开始追问 |
| 希望/想要/期待/要是能 | 层5-期待 | 从期待层开始追问 |
| 没人懂我/没人理解/孤独 | 层7-自我 | 可能涉及身份认同危机 |
| 无聊/没意义/空虚 | 层6-渴望 | 可能在寻找深层意义 |

### 追问序列生成

**情况A: 从层1开始（行为触发）**
```
step 1: "你应对这个情况的方式是什么？" → 收集层2信息
step 2: "你当时有什么感受？" → 收集层3信息（基于step1的应对方式推断）
step 3: "你在想什么？" → 收集层4信息
step 4: "你当时希望发生什么？" → 收集层5信息
step 5: "你内心深处在渴望什么？" → 收集层6信息
step 6: "这让你觉得自己是什么样的人？" → 收集层7信息
```

**情况B: 从层3开始（感受触发）**
```
step 1: "你应对这种感觉的方式是什么？" → 收集层2信息
step 2: "是什么让你有这种感受？" → 收集层4信息（追到观点/解读层）
step 3: "你当时希望怎样？" → 收集层5信息
step 4: "这种感觉背后你在渴望什么？" → 收集层6信息
step 5: "这种感觉让你觉得自己是谁？" → 收集层7信息
```

**情况C: 从层4开始（观点触发）**
```
step 1: "这种想法让你产生了什么感受？" → 收集层3信息
step 2: "为了应对这个，你做了什么？" → 收集层2信息
step 3: "你内心真正想要的是什么？" → 收集层6信息
step 4: "这个想法背后，你认为自己是什么样的人？" → 收集层7信息
```

### 追问话术库

**探测层2（应对方式）：**
- "你当时做了什么？"
- "面对这件事，你是怎么处理的？"
- "你采取了什么行动？"

**探测层3（感受）：**
- "那一刻你心里是什么感觉？"
- "你当时的情绪是什么？"
- "如果给这个感受起个名字，你会叫什么？"

**探测层4（观点）：**
- "你在想什么？"
- "当时脑子里出现了什么想法？"
- "你如何看待这件事？"

**探测层5（期待）：**
- "你本来希望事情怎么发展？"
- "你期望的结果是什么？"
- "如果一切如你所愿，会是什么样的？"

**探测层6（渴望）：**
- "你内心深处真正想要的是什么？"
- "这种感觉背后，你最渴望的是什么？"
- "如果不需要考虑现实，你最想要什么？"

**探测层7（自我）：**
- "这让你觉得自己是什么样的人？"
- "这件事定义了你对自己的什么看法？"
- "它影响了你是谁的认知吗？"

### Coach 联动说明

当冰山追问完成后（达到层5以上），输出结构化诊断：

1. **如果层6-层7信息丰富** → 用户可能在经历深层的身份/意义危机，Coach 应聚焦存在性议题
2. **如果层2应对方式单一** → 用户可能缺乏灵活的应对策略，Coach 应教应对技能
3. **如果层3感受强烈但层4观点理性** → 用户可能有情感隔离，Coach 应帮助情绪整合
4. **如果层5期待与层7自我冲突** → 用户可能在经历价值观矛盾，Coach 应协助澄清
