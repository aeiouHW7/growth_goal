# 每次对话自动加载（按顺序拼接）

## 第一层：最近7天的碎碎念

读取 `memories/daily_raw/` 中最近7个文件，提取【内容】和【我的洞察】字段。

**加载格式：**
```
## 近7天记忆摘要

【YYYY-MM-DD】
内容：...
洞察：他没意识到的是...

【YYYY-MM-DD】
内容：...
洞察：...
```

**Token限制：**
- 总 tokens 不超过 2000（约 1500 汉字）
- 超长时优先裁剪早期碎碎念的"内容"，保留"洞察"
- 确保压缩后仍有足够上下文做判断

---

## 第二层：上次未解决的"待追踪"

从最新的 `memories/weekly_summaries/` 或 `memories/monthly_reports/` 中扫描 `## 待追踪` 段落。

**加载格式：**
```
## 待追踪问题

[日期] 问题描述 — 状态：待追问/处理中
```

按时间排序（最久的在前）。

---

## 第三层：最低分三维度

读取 `config/capability_baseline.json` 中的 `lowest_three_dimensions`。

**加载格式：**
```
## 能力短板（最近一次评分）

1. [维度名]：X分 — [简述问题]
2. [维度名]：X分 — [简述问题]
3. [维度名]：X分 — [简述问题]

涉及这些维度的碎碎念必须进入"挑战模式"
```

---

## 第四层：长期模式（始终加载）

读取 `memories/long_term/` 下所有 .md 文件的标题和第一句话。

**加载格式：**
```
## 已知模式

- [模式名称]：首次发现 [日期]，当前状态：[状态]
- ...
```

---

## 加载完整性检查

每次对话开始前，自检：

- [ ] 近7天记忆是否加载？
- [ ] 待追踪问题是否加载？
- [ ] 能力短板是否加载？
- [ ] 长期模式是否加载？

如果任何一个没做，这条对话就是**失忆的**，后续判断可能出错。
