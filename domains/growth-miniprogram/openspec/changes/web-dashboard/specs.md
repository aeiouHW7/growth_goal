# Web Dashboard 详细规格

## 1. 主题与全局样式

### 浅色主题
- 背景色: #f5f5f7
- 卡片: #ffffff, border: #e5e7eb, border-radius: 12px
- 文字主色: #111827
- 文字辅助: #6b7280
- 强调色: #6366f1 (indigo)
- 成功: #22c55e
- 警告: #eab308
- 错误: #ef4444
- 缺卡: #9ca3af

---

## 2. 导航规格

### 顶部导航栏
- 左侧: "growth" logo
- 右侧: 三个导航标签，点击切换内容区
  - 总览 → OverviewPage
  - 目标链 → GoalsPage
  - 计划 → PlansPage
- 选中标签: 紫色文字 + 浅紫背景 (#6366f1 / #eef2ff)
- 未选中: 灰色文字 (#6b7280) + 白底

### 页面默认
- 首次加载: 显示 "总览" 标签
- 无数据时显示空状态

---

## 3. 总览页规格 (OverviewPage)

### 用户信息
- 右上角: 紫色渐变圆形头像 + 首字母
- hover 弹出卡片:
  - 昵称、职业、行业
  - 可用时间（工作日/周末）
  - 关注领域列表

### 人生总目标
- 卡片展示
- 标题 + 时间跨度 (10-20年)
- 描述文本
- 无操作按钮，纯展示

### 进行中的年度目标
- 只展示状态为"进行中"的年度目标，已完成目标不展示
- 每项: 颜色圆点(绿/黄) + 标题 + 进度百分比 + 进度条
- 右上角显示年份

### AI 建议改进点
- 左侧彩色竖线分区:
  - 绿色: 正向反馈/保持
  - 黄色: 警告/需改进
  - 红色: 严重问题
- 内容来源: 日复盘、周复盘、月复盘的 AI 分析汇总

### 当月目标完成进度
- 标题显示当前月份
- 每个目标一行:
  - 目标名称
  - 当月进度值 (如 5.2万 / 6万)
  - 进度条
- 无年度汇总、无筛选器

---

## 4. 目标链页规格 (GoalsPage)

### 筛选栏
- 三个筛选按钮: 全部 | 进行中 | 已完成
- 选中: 紫色文字 + 浅紫背景
- 未选中: 灰色文字 + 白底

### 目标层级树
```
LifeGoal
├── 标题 + 状态标签 + 时间跨度
├── YearlyGoal (缩进)
│   ├── 标题 + 年份 + 状态标签 + 进度条 + 进度值
│   └── MonthlyPlan (更深缩进)
│       ├── 标题 + 月份 + 进度条 + 进度值
```

### 交互
- 点击展开/折叠子目标
- 已完成目标: 整体置灰 (opacity: 0.45)，文字变灰，进度条灰色

---

## 5. 计划页规格 (PlansPage)

### 日期选择器
- 年份下拉: 2026, 2027, ...
- 月份下拉: 1月 ~ 12月

### 维度切换
- 四个按钮: 年 | 月 | 周 | 日
- 选中: 紫色文字 + 浅紫背景

### 5.1 月视图 (CalendarGrid)
- 7 列 (周一~周日)
- 正确计算每月第一天偏移
- 调用 `GET /api/progress/calendar?year=&month=`
- 颜色编码圆点:
  - 🟢 优秀 (analysisScore >= 80)
  - 🟡 良好 (analysisScore >= 60)
  - 🔴 差 (analysisScore < 60)
  - ⚪ 缺卡 (有复盘无评分)
- 图例
- 今日: 蓝色边框高亮

### 5.2 周视图 (WeekTimeline)
- 标题: "第N周 (M/D - M/D)"
- 每日一行: 周X + 日期 + 内容摘要 + 评级圆点

### 5.3 日视图 (DayTimeline)
- 标题: "YYYY年MM月DD日 周X"
- 时间线: 按时间块显示可用时段
- 日计划列表 + 复盘摘要

### 5.4 年视图 (YearGrid)
- 4列 x 3行 = 12个月块
- 每月块: 月份名称 + 评级表情符号 + 完成百分比

### 右侧三面板 (通用)

#### 计划面板
- 当前周期计划列表
- 每项: 颜色圆点 + 标题 + 进度

#### 评估面板
- 评级表情符号 (🟢/🟡/🔴)
- 综合评分 + 完成率

#### 报告面板
- 4 个区块: 完成亮点 / 偏差问题 / 诊断建议 / 外部视角

---

## 6. 空状态规格

### 触发条件
- 数据库无 User
- 或无任何 Goal

### 显示
- 居中: "还没有设定目标"
- 说明: "在终端输入 growth，开始设定你的目标吧！"
- CTA: "打开终端 → 输入 growth → 选择 2 管理目标"

---

## 7. 加载与错误状态

### 加载
- 首次: "加载中..."
- 切换周期: 仅刷新数据区域

### 错误
- API 超时/失败: "数据加载失败" + 重试按钮
- 特定面板无数据: "暂无数据" 灰字

---

- 端口: 3002

- 无外部 UI 依赖
