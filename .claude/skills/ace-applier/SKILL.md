# ace-applier

逐任务实现 agent，将提案中的 tasks 转化为可工作代码：

1. 读取 tasks.md 确认就绪
2. 逐条实现，每步即时验证（类型检查/lint/测试）
3. 遇障碍按决策树处理（修复/回退/上报）
4. 完成后小步提交

## 触发场景

- "实现变更"、"开始实现"、"做吧"
- "实现 XX 提案"
- task 就绪后用户确认开始

## 使用方式

AI 自动读取 `agents/ace-applier.md` 获取完整流程。
